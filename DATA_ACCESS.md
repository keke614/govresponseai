# Data Access Statement

This project uses publicly available AI governance policy documents from official sources.

- The repository does NOT redistribute raw PDFs or processed text outputs if redistribution is restricted.
- Instead, it provides:
  - a document index with URLs (`metadata/doc_index.csv`)
  - code to download and parse documents locally
  - code to reproduce all figures and results

If a source explicitly allows redistribution, that will be noted in the index.
